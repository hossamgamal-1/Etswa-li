Etswa’li
Etswa’li is an e-commerce application with many features:
- Authentication (email&password and phone number)
- Responsive user interface
- Favourite List
- Cart List
- Searching
- Tracking Checkedout Orders (Under Progress Feature)
Developed using Dart, Flutter, SharedPrefrences, Rest Api, Provider and Firebase
